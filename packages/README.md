# Packages

Narimato does not vendor GDS under `packages/gds-*`.

Install design-system packages from the SSOT release or npm registry:

- **Repo:** [sovereignsquad/general-design-system](https://github.com/sovereignsquad/general-design-system)
- **Scope:** `@doneisbetter/gds-theme`, `@doneisbetter/gds-core`, `@doneisbetter/gds-admin`
- **Current pin:** [gds-v2.6.1](https://github.com/sovereignsquad/general-design-system/releases/tag/gds-v2.6.1) (see `package.json`)

See [docs/GDS_ADOPTION.md](../docs/GDS_ADOPTION.md) and [gds-adoption.json](../gds-adoption.json).
